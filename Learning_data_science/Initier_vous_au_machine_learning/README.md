# Initier Vous au Machine Learning

Cette partie contient mes notes, exercices et expérimentations liés au cours [Initiez-vous au Machine Learning](https://openclassrooms.com/fr/courses/8063076-initiez-vous-au-machine-learning) proposé par OpenClassrooms.

## Objectif

- Découvrir les grands principes du Machine Learning

- Manipuler les fonctions de base d'un modèle prédictif

- Transformer des jeux de données

- Optimiser les performances d’un modèle

## Outils utilisés

- Python 3.14+
- Google Colab (recommandé pour une exécution rapide)
- Jupyter Notebooks
- Visual Studio Code (En Local)

- Partie 1 : Découvrez les grands principes du Machine Learning

- Partie 2 : Manipulez les fonctions de base d'un modèle prédictif

- Partie 3 : Transformez vos jeux de données

- Partie 4 : Optimisez les performances d’un modèle
