# La modélisation statistique

On est dans une démarche d'analyse et d'interpretation de la dynamique entre les variables.
Elle s'appuie sur des test d'hypothèses et des modèles mathématiques pour évaluer les conclusions de l'analyse.

exemple de question : **quel est le profil des utilisateurs qui s'abonnent?**


# La modélisation prédictive

L'approche machine learning .
On attend du modèle prédictif :

- des prédictions de qualité ( soit un bon score calculé grâce à une métrique choisie au préalable)

- la robustesse (capacité à extrapoler soit à généraliser à partir des données d'entrainement)

exemple de question : **Comment prédire qu'un nouvel utilisateur va s'abonner ?**

La logique de prédiction du modèle est assimilé à une boîte noir et
la question d'interprétabilité du modèle ne se pose pas en général mais des solutions existent pour
déterminer quelles sont les variables qui ont le plus influencées une prédiction.

Des librairies permettent d'expliquer la prédiction de la plupart des modèles de ML (Machine Learning) :

- [Shap](https://github.com/shap/shap)

- [Lime](https://github.com/marcotcr/lime)

# Différence entre modélisations statistique

|              |   Modélisation statistique    |   Modélisation prédictive   |
|   ---        |         ---                   |            ---              |
|   Objectif   |     Expliquer et Analyser     |           Prédire           |
|   Focus   |  Fiabilité des conclusions : test d'hypothèses et intervalles de confiance    |  Performance , résilience et robustesse du modèle  |
| Utilisation des données | Tous les échantillons | Une partie pour l'entrainement et une partie pour l'evaluation du modèle |
| Librairies python | [statsmodel](https://www.statsmodels.org/stable/index.html)  | [scikit-learn](https://scikit-learn.org/stable/) , [PyTorch](https://pytorch.org/) , [TensorFlow](https://www.tensorflow.org/)

# Comprenez en quoi consiste le Machine Learning
Le but du ML est d'entrainer un modèle prédictif à partir d'un jeu de données.

On distingue la variable cible des autres variables potentiellement prédictrices.

Voici les étapes de construction d'un modèle prédictif à partir d'un dataset

- Préparation des données

- Repartition des données entre entrainement , test (Validation croisée)

- Optimisation


# Les approches du ML

On distingue :

- Approche supervisée : classification quand on prédit des catégories et regression quand on prédit des variables continues.

- Approche non supervisée : clustering
